# CASTILE MOBILE BOT

Castile Mobile Bot adalah bot Telegram yang memungkinkan pengguna menonton iklan untuk mendapatkan saldo tambahan, melakukan withdraw, dan melacak riwayat transaksi. Castile Mobile Bot ini menggunakan Telegram untuk memberikan pengalaman interaktif kepada pengguna.

## Fitur Utama

- Tonton Iklan          : Pengguna dapat menonton iklan dan mendapatkan saldo tambahan.
- Withdraw Saldo        : Pengguna dapat menarik saldo mereka secara langsung.
- Riwayat Transaksi     : Menampilkan riwayat transaksi terbaru pengguna.
